# I_T_iOSPractice

iOS ゲームアプリの個人研究・検証用のプロジェクトです。

下記について検証などを行なっています。
- Visutal Studio for Mac と OpenTK を利用した描画処理
- Xcode で生成した静的ライブラリの組み込み
- fastlane を利用したアプリ開発フローの自動化
- Jenkins を利用したアプリ開発フローの自動化
